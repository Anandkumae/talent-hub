# **App Name**: Internal Talent Hub

## Core Features:

- User Authentication and Roles: Secure user login/signup with role-based access control (Admin, HR, Manager, Candidate).
- Job Management: Create, edit, and manage job postings with details such as title, department, description, and requirements.
- Candidate Management: Upload, store, and manage candidate resumes, including updating candidate status (Applied, Shortlisted, Interviewed, Hired, Rejected).
- Resume Upload: Allow candidates to upload resumes (PDF, DOC) that are stored securely, and visible in candidate details.
- Dashboard: Provide an overview dashboard with key metrics such as total jobs posted, total candidates, and count by hiring stage.
- AI Resume Matcher Tool: Automatically match candidate resumes to job descriptions and provide a match score out of 100, to assist HR with prioritizing the best candidates.

## Style Guidelines:

- Primary color: HSL(210, 60%, 50%) converted to RGB Hex: #478AE6 to reflect professionalism and trust.
- Background color: Light grey HSL(210, 15%, 95%) converted to RGB Hex: #F2F4F7, to provide a clean and uncluttered backdrop.
- Accent color: HSL(240, 40%, 60%) converted to RGB Hex: #7587D9 for highlighting interactive elements and key information.
- Body and headline font: 'Inter' sans-serif for a modern, neutral, and readable experience.
- Use a set of simple, professional icons to represent different actions and modules within the application. Icons should maintain a consistent style and follow accessibility best practices.
- The layout will be clean and corporate-like, featuring a sidebar navigation for accessing different modules and table-based lists for job and candidate management. Summary metrics are displayed using cards on the dashboard.
- Subtle animations and transitions will be used to enhance the user experience without being distracting.
